# Phase 3 Manual Review Checklist

Manual review of technique descriptions and instruction steps. Prioritize techniques with curated content (used in lessons).

## Review Criteria

- **longDescription**: Does it correctly describe the technique? Clear for a learner? Any jargon?
- **shortDescription**: Matches the technique? Concise and accurate?
- **Step text**: Do rendered steps correctly describe the pattern? Highlights correct? Any confusing wording?
- **Mochi hints**: Appropriate and on-brand?

## Status Legend

- **pending**: Not yet reviewed
- **reviewed**: Reviewed, no changes needed
- **needs-fix**: Issues found, needs update

## How to Review Step Text

1. Run the app and open a technique lesson.
2. Step through each instruction.
3. Verify the text matches the highlighted cells and the pattern being taught.
4. Check that helper-extracted values (e.g. "row 3", "candidates 1 and 8") are correct for the puzzle.
